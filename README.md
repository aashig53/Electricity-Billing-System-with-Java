This project presents an Electricity Billing System designed to facilitate the management of electricity bills for consumers.
Developed using Java, Swing, and MySQL, the system offers a user-friendly interface for tasks such as bill payment, data retrieval, and account management. 
Users can input their details, view their billing information, and make payments (faux). 
Overall, this Electricity Billing System streamlines the billing process, providing convenience and efficiency for both consumers and service providers.
